# CinemarX-M-dulo-5
Este módulo forma parte del ecosistema CinemarX, una aplicación desarrollada en Java para la gestión de un cine. El módulo M5 se encarga del flujo crítico sobre pago y finalización de compra, permitiendo a los usuarios interactuar con sus métodos de pago guardados o registrar nuevas tarjetas bancarias en tiempo real.
